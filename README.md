# LOFTY GARDEN

## Prerequisite
- Local router
- Electricity
- Arduino (UNO)
- Raspberry

## Server

Install :
> npm install

Run :
> node index.js

### API
 To call API backend
 
 > nodemon api.js

## Arduino
Contain Arduino sensor and wifi code. Import it in Arduino IDE and export it to your arduino.
